<h1>DeepL API Component</h1>
<h3>Desctiption: DeepL API Utility Bar Component for a fast and easy to use translation tool in Salesforce org.</h3>
<h4>Features:</h4>
  <li>Source language selection including automatic detection</li>
  <li>Target language selection</li>
  <li>Formality options</li>
  <li>Ability to open it in a pop-up window</li>
  <li>Some support for formatted text (for example, when copy/pasting a value from a Salesforce Rich Text field)</li>
  <li>Available languages: BG, CS, DA, DE, EL, EN, EN-GB, EN-US, ES, ET, FI, FR, HU, IT, JA, KO, LT, LV, NB, NL, PL, PT, PT_BR, PT-PT, RO, RU, SK, SL, SV, TR, UK, ZH</li>
  <li>File translation (both local files and files stored in Salesforce) File formats: docx, pptx, pdf, txt</li>
  <li>Assignment notes: Use a free DeepL plan for this assignment, use DeepL API docs</li>
